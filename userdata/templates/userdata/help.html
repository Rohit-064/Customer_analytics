Steps to setup the project
Localhost
1.Create virtualenv
2.In cmd> virtualenv -p python3.6 < extracted Customer_analytics >
3.cmd> cd Customer_analytics
4.cmd> scripts/activate
5.cmd> pip install -r requirements.txt
6.cmd> python manage.py runserver
7.Go to http://127.0.0.1:8000/userapp in browser
Heroku
1.Go to https://dashboard.heroku.com/apps > New > Create new app
2.Enter app_name > create app
3.Login to Your Github account > Add New repository > Give repository name > add downloaded folder to newly created repository
4.Go to Deployment method > Github > Connect to Github > Enter Repo Name > Search > Connect
5.Go to Manual deploy > Deploy Branch / Automatic deploys can be enabled
6.Go to app_name.herokuapp.com/userapp
